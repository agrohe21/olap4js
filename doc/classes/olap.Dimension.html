olap.Dimension Class

Defined in: olap.js:715
Module: olap

An organized hierarchy of categories, known as levels, that describes data in a cube. A Dimension typically describes a similar set of members upon which the user wants to base an analysis A Dimension must have at least one Hierarchy, and may have more than one.
