olap.Cube Class

Defined in: olap.js:389
Module: olap

A Cube is the central metadata object for representing multidimensional data. It belongs to an olap.Schema, and is described by a list of dimensions and a list of measures. It may also have a collection of named sets, each defined by a formula.
