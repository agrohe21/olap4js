olap.Datasource Class

Defined in: olap.js:157
Module: olap

A Datasource is the highest level element in the hierarchy of metadata objects. A database contains one or more catalogs. Some OLAP servers may only have one database. Mondrian is one such OLAP server. To obtain the collection of databases in the current server, call the olap.Connection.getOlapDatabases() method.
